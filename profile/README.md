## About

DataLab Rotterdam is a pioneering sector within **Hogeschool Rotterdam** (Rotterdam University of Applied Sciences), focused on data infrastructure, cloud services, and artificial intelligence. We build real platforms and run real research projects, giving students hands-on experience with production-grade infrastructure while pushing the boundaries of what's possible with AI, IoT, and cloud technology.

Where innovation meets simplicity.

## What We Build

| Service | Description |
|---|---|
| **Atlas** | Integrated platform consolidating our services |
| **Identity** | Authentication & access management |
| **[Nova AI](https://www.linkedin.com/showcase/dl010-nova-ai)** | AI / machine learning infrastructure, our latest innovation |
| **Pulsar IoT** | IoT device management |
| **Spaces** | Virtual machine & cloud hosting |

Our infrastructure includes GPU-backed compute, time-series databases, and data visualization tooling, designed to integrate seamlessly across the stack.

## Research & Projects

We work on a broad range of applied research and engineering projects, including:

- 🖐️ Gesture recognition using browser-based machine learning
- 🚢 Maritime AI applications, developed with external industry partners
- 🌐 Digital twin technology
- 🔐 Educational platforms for teaching API security
- 🛒 Accessibility-focused e-commerce systems
